#RN02

Somente alunos matriculados podem entregar atividades.

#RN03

Atividades possuem prazo de entrega.

  #Impacto da RN03

  #User Story

  Como estudante, desejo visualizar o prazo das atividades para realizar minhas entregas dentro do período permitido.

  #BPMN

  Verificação automática de prazo.

  #Arquitetura

  Serviço de validação de datas.

  #RNF

  Confiabilidade.

#RN04

Após o prazo, a entrega fica bloqueada.

#RN05

Professores podem atribuir notas apenas às suas turmas.

#RN06

Cada aluno pode enviar apenas uma versão final da atividade.

#RN07

Notificações devem ser enviadas quando uma atividade for publicada.

#RN08

Apenas administradores podem cadastrar novos professores.

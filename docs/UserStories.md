User Stories alinhadas ao BPMN
US01

Como aluno, desejo acessar uma atividade para visualizar suas instruções.

US02

Como aluno, desejo realizar uma atividade para concluir minhas tarefas acadêmicas.

US03

Como aluno, desejo enviar minha atividade para receber avaliação do professor.

US04

Como professor, desejo visualizar os envios realizados pelos alunos para iniciar a correção.

US05

Como professor, desejo corrigir atividades para avaliar o desempenho dos estudantes.

US06

Como professor, desejo atribuir notas para registrar o resultado da avaliação.

US07

Como professor, desejo registrar feedback para orientar os alunos sobre seu desempenho.

US08

Como professor, desejo ser notificado quando uma atividade for enviada.

US09

Como aluno, desejo ser notificado quando minha nota estiver disponível.

US10

Como aluno, desejo visualizar minha nota e feedback após a correção.
